# Recipe Generator with Claude AI 🍳

A Python application that generates detailed, authentic recipes using Anthropic's Claude AI.

## Features

✨ **Smart Recipe Generation**
- Generate custom recipes for any dish
- Specify number of servings
- Add dietary preferences (vegetarian, spicy, mild, etc.)
- Get detailed ingredients with measurements
- Step-by-step cooking instructions

🔒 **Secure**
- API key stored in environment variables
- Never exposed in code or version control

🎯 **User-Friendly**
- Simple command-line interface
- Clear error messages
- Generate multiple recipes in one session
